# SocioSim

SocioSim is a deterministic synthetic scenario simulator for policy, moderation,
feed, advertising, and audit-workflow experiments.

The non-negotiable boundary is simple: synthetic assumptions must never be
presented as measured evidence. Outputs are scenario diagnostics, not estimates
of real-world performance.

## Install

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -e .
pip install -e .[dev,e2e,security,asset]
```

## Run

```bash
python run.py
python run.py --web
python run.py --profile aggregate_matched_prototype
python run.py --classifier synthetic_template_classifier
python run.py --measure-classifier
```

Classifier modes are synthetic mechanics modes:

- `synthetic_noise_classifier`
- `synthetic_template_classifier`

No runtime classifier mode is a real deployable classifier. Component benchmark
diagnostics apply only to the benchmark artifact generated by
`python run.py --measure-classifier`.

## Evidence Contract

Machine-readable evidence records live in:

- `socio_sim/data/evidence_registry.json`
- `socio_sim/data/scenario_assumptions.json`

Metric payloads include provenance metadata: evidence ID, evidence kind,
verification status, valid uses, invalid uses, and limitations.

Current built-in behavior, persona, advertising, profile, policy-timing, content
rate, and classifier operating-point defaults are scenario assumptions unless a
user supplies evidence. Legacy aggregate target files are aggregate-fit
diagnostics only because their source metadata is incomplete.

## Assets

The dashboard uses v4 direct-use assets under:

`socio_sim/web/static/assets/v4/`

These are project-owned synthetic decorative PNGs. They are not documentary
images, screenshots, testimonials, product claims, KPIs, or evidence.

Run asset QA with:

```bash
python scripts/asset_qa.py
```

## Verification

```bash
python -m ruff check socio_sim tests run.py
python -m pytest -q --cov=socio_sim --cov-report=term-missing --cov-fail-under=85
python -m pytest -q tests/test_e2e_playwright.py
python scripts/evidence_gate.py
python scripts/claim_scan.py
python scripts/secret_scan.py
python scripts/asset_qa.py
python run.py --measure-classifier
python run.py --backtest
```

`python -m build --wheel` requires the `build` package and should be run from an
environment where the package is installed.

## Use Boundaries

Do not use SocioSim outputs to target, rank, enforce against, or predict real
people, organizations, platforms, campaigns, or public events. Policy mappings
are research approximations and are not legal advice.
